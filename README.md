# 刘靖 - 个人简历网站

## 简介

这是一个现代、专业的响应式个人简历网站，展示刘靖先生24年国际贸易领域的丰富经验。

## 特点

- ✨ 现代化深色主题设计
- 📱 完全响应式布局（适配手机、平板、桌面）
- 🎨 流畅的动画效果和渐变背景
- 🚀 单页面应用体验
- ⚡ 快速加载，无需后端

## 技术栈

- HTML5
- CSS3 (渐变、动画、Flexbox、Grid)
- JavaScript (滚动动画、交互效果)
- Font Awesome 图标

## 部署方式

### 方式一：Vercel（推荐）

1. 将代码推送到 GitHub
2. 登录 [Vercel](https://vercel.com)
3. 导入项目并一键部署

### 方式二：Netlify

1. 登录 [Netlify](https://netlify.com)
2. 拖拽文件夹上传

### 方式三：GitHub Pages

1. 创建 GitHub 仓库
2. 上传代码
3. 启用 GitHub Pages
